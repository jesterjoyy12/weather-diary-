# Weather Diary (Дневник погоды)

**Автор:** Ксения Шканова  
**Дата:** Апрель 2026

## Описание программы
Weather Diary — это desktop-приложение для ведения личного дневника погоды.  
Позволяет добавлять, удалять, фильтровать записи о погоде и сохранять их в JSON.

## Возможности
- Добавление записи: дата, температура, описание, осадки
- Фильтрация:
  - по точной дате
  - по температуре (выше заданного значения)
- Удаление выбранной записи
- Сохранение и загрузка из `weather_data.json`
- Валидация ввода (дата, температура, непустое описание)

## Как запустить
1. Установите Python 3.6+
2. Скачайте файл `main.py`
3. В терминале выполните:
```bash
python main.py
